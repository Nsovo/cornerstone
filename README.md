# Cornerstone React Website — Yarn

A responsive React + TypeScript website concept for Cornerstone Open Portal Network.

## Requirements

- Node.js 20 or newer
- Yarn Classic 1.x

If Yarn is not installed:

```bash
corepack enable
corepack prepare yarn@1.22.22 --activate
```

## Run locally

```bash
yarn install
yarn dev
```

Open the local URL shown by Vite.

## Production build

```bash
yarn build
yarn preview
```
